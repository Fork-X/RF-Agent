# QuanGan 系统架构总览

## 概述

QuanGan-py 是基于 Python 3.12+ 的多智能体 CLI AI 助手，采用以下核心设计理念：

- **ReAct 循环推理**：基于思维-行动-观察的迭代推理模式
- **双层记忆系统**：Core Memory（长期）+ Life Memory（短期），详见 [记忆系统架构](memory-system.md)
- **多 Agent 协作**：主 Agent 协调多个专业子 Agent 完成任务
- **上下文自动压缩**：智能摘要机制保持对话上下文在可控范围
- **动态 Skill 加载**：基于触发词的自动技能激活与提示注入

## 关键模块职责

| 模块 | 路径 | 核心职责 |
|------|------|----------|
| **CLI** | `src/quangan/cli/` | REPL 循环、命令路由（/help, /clear, /skills 等）、会话持久化、UI 显示（spinner、token 用量） |
| **Agent** | `src/quangan/agent/agent.py` | ReAct 推理循环、工具执行编排、中断控制（ESC 键）、上下文压缩触发、Skill 管理 |
| **Sub-Agents** | `src/quangan/agents/(coding\|daily)/` | 无状态子 Agent 工厂、任务特定工具集、独立迭代限制 |
| **LLM** | `src/quangan/llm/` | 协议抽象（OpenAI/Anthropic）、API 通信、流式响应、工具调用解析 |
| **Memory** | `src/quangan/memory/` | 双层存储（Core/Life）、强化计数、Life→Core 自动整合、记忆检索 |
| **Skills** | `src/quangan/skills/` | 动态加载、触发检测、系统提示注入、三级搜索路径 |
| **Config** | `src/quangan/config/` | 提供商管理（DashScope/Kimi/OpenAI/Anthropic）、环境变量加载、模型配置 |

## 端到端流程演示

### 场景一：代码修改任务

```
用户输入: "帮我修改 src/utils.py，添加一个日志函数"
    │
    ▼
┌─────────────────────────────────────────┐
│ CLI 解析输入，识别非命令，传递给主 Agent │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ Skill 触发检查（如 "python" 相关 Skill） │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ 主 Agent.run() 启动 ReAct 循环          │
│ - 添加用户消息到历史                     │
│ - 构建工具定义列表                       │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ LLM 决策：需要调用 coding_agent          │
│ 返回 tool_call: coding_agent            │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ 主 Agent 执行 coding_agent_handler      │
│ - 创建新的 Coding Agent 实例（无状态）   │
│ - 传递任务参数                           │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ Coding Agent 独立 ReAct 循环            │
│ ┌─────────┐ ┌─────────┐ ┌─────────┐    │
│ │read_file│→│edit_file│→│verify_  │    │
│ │(utils.py)│ │(添加函数)│ │ code    │    │
│ └─────────┘ └─────────┘ └─────────┘    │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ 子 Agent 返回执行结果给主 Agent          │
│ "已成功添加日志函数到 src/utils.py"      │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ 主 Agent 生成最终响应                    │
│ "已完成修改，添加了 log_info() 函数..."  │
└─────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────┐
│ CLI 显示响应 + Token 用量                │
│ save_session() 持久化对话历史            │
└─────────────────────────────────────────┘
```

└─────────────────────────────────────────┘
    │
    ▼

    ▼
┌─────────────────────────────────────────┐

    ▼
